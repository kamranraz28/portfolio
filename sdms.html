Smart Diagnostic Management System

Project Info

A smart and user friendly software for the Lab Quest Limited, a popular diagnostic center in Bangladesh.

A smart software to make easier the procedure to book a health package or book an appointment of a doctor for it's user. Doctor can set the slot by himself that when in a day he is available to book for an appointment. Patient can book a slot from that given slot by Doctor. After visiting the patient Doctor can prescribed the patient on the software and the patient history will be stored to the system so that next time when this patient will come again, the doctor can see the complete patient history from the software.

As well as after doctor prescribing the patient on software, the patient will be able to download as a PDF this prescription by logging into his account.

Panels:
  1. Admin Panel
  2. Visitor Panel
  3. Patient Panel
  4. Doctor Panel
  5. Call Center Panel
Key Features:

Admin Panel:
  1. Add Doctor
  2. Add different levels of user
  3. Can set the slot for the doctor
  4. Can prescribe for the doctor
  5. Various reporting
Doctor Panel:
  1. Set the slot of the schedule
  2. Patient history
  3. Online prescription
Patient Panel:
  1. Booking health package
  2. Booking appointment of doctors
  3. Online prescription
Call Center Panel:
  1. Upcoming patient list
  2. Book doctors for those patients who are booking over the phone
  3. Cancel appointment

Client

Lab Quest Diagnostic Center, North Badda, Dhaka, Bangladesh

Skill Used

PHP, Laravel, JavaScript, MySQL
